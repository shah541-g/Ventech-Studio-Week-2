📦 CSS overflow Property — Active Recall Notes

Video: visible, hidden, clip, scroll, auto

1. Overflow Kya Hai?

overflow property define karti hai ke jab kisi element ka content uske parent box mein fit nahi hota (yani overflow hota hai), to browser kya behavior dikhaye.

div {
  border: 2px solid;
  height: 75px; /* content is se zyada ho jaaye to overflow */
}

5 possible states hain: visible, hidden, clip, scroll, auto.

2. overflow: visible (Default)

Default value. Agar content overflow ho raha ho, to usko bahar dikhne diya jaata hai — box ke border se bahar visible rehta hai, koi apparent restriction nahi.

3. overflow: hidden

Overflow hone wala content chhupa (hidden) diya jaata hai — screen par nazar nahi aata.

⚠️ Important: content hidden hone ka matlab ye nahi ke wo gayab ho gaya — wo DOM mein maujood aur interactable hai. Jaise agar aap select-all + copy karo, to poora text (including hidden part) copy ho jaayega.

4. overflow: clip

hidden jaisa hi visually — content overflow hoke chhup jaata hai. Farq ye hai ke clip ka use overflow-clip-margin property ke sath tandem mein hota hai.

div {
  overflow: clip;
  overflow-clip-margin: 13px;
}

overflow-clip-margin se define hota hai ke content box se kitna bahar tak dikh sakta hai before wo cut ho jaaye — is example mein 13px tak overflow allowed hai, baaki hidden.

5. overflow: scroll

Box par hamesha scrollbar (vertical aur horizontal, dono) dikhti hai, chahe content overflow ho ya na ho.

⚠️ Agar content itna kam ho ke overflow ho hi na, to scrollbar phir bhi dikhti hai — bas wo disabled (non-functional) state mein rehti hai.

6. overflow: auto

scroll jaisa hi, lekin smart — scrollbar sirf tab dikhti hai jab actually zaroorat ho (content overflow ho raha ho). Agar overflow na ho, to scrollbar bilkul nahi dikhti.
