📏 CSS Units — Active Recall Notes

Video: px, %, vw/vh, em/rem, vmin/vmax

1. Absolute vs Relative Units

TypeUnitsBehavior
Absolute Fixed px, pt, cm, mm Fixed values — screen size chahe kuch bhi ho, change nahi hote
Relative Dynamic %, em, rem, vw, vh Kisi doosri value (parent font-size, screen size waghera) ke basis par calculate hote hain

px simple hone ki wajah se bohot use hota hai (1px = ek screen dot), lekin modern responsive design mein relative units zyada useful hote hain.

2. Percentage (%)

Element ki size uske parent ke relative calculate hoti hai.

.box-1 { width: 50%; }   /* parent ka aadha width */
.box-2 { width: 100px; } /* fixed, parent se koi lena dena nahi */

Agar parent 500px wide ho, to width: 50% wala box 250px ban jaata hai, jabke width: 100px wala box hamesha 100px hi rahega — parent chahe jitna bhi bada/chhota ho.

⚠️ Agar parent ki width change ho, to percentage-based box automatically resize ho jaata hai (nayi width ka wahi percentage banaye rakhta hai) — isliye percentage responsive layouts ke liye acha hai.

3. box-sizing: border-box

Agar element ki width 100% (ya koi fixed value) set ho aur usme border bhi ho, to horizontal scrollbar aa sakta hai (kyunke border width se bahar add hota hai).

* {
  box-sizing: border-box;
}

Isse border aur padding defined width ke andar hi included ho jaate hain, aur unwanted scrollbar khatam ho jaata hai.

4. vw / vh (Viewport Units)

vw = viewport width, vh = viewport height. Ye screen ke current visible size ke relative hote hain.

.card {
  width: 100vw;  /* poori screen ki width */
  width: 80vw;   /* screen width ka 80% */
  height: 50vh;  /* screen height ka 50% */
}
⚠️ 100vw kabhi kabhi horizontal scrollbar create kar sakta hai (agar scrollbar khud kuch space le raha ho) — isliye zaroorat ke hisaab se thora kam value (jaise 80vw) use karna behtar ho sakta hai.

Fixed pixel values ke bajaye vw/vh use karne se layout har screen size par automatically adjust hota hai — pixel-based layout chhoti screens (jaise 1920px monitor) par break ho sakta hai.

5. margin: auto (Centering)

.card {
  margin: 20px auto;
}

Format: top-bottom auto-left-right. Isse element horizontally center ho jaata hai, aur top/bottom par 20px space bhi milti hai.

⚠️ margin: auto se centering sirf block-level elements par kaam karti hai — agar element inline hai, to ye center nahi hoga.

6. em vs rem

em — Parent Ke Relative

.paragraph {
  font-size: 1em; /* parent ke font-size ka 100% */
}
  • Container ka font-size 16px ho, aur paragraph 1em use kare → paragraph bhi 16px hoga (kyunke 1em = parent size ka 100%)
  • Container ka font-size 20px ho → 1em wala paragraph 20px ho jaayega
  • 2em = parent ke font-size ka double (agar parent 20px ho, to paragraph 40px)
  • Agar parent par koi font-size set na ho, to browser ka default 16px use hota hai (isliye 2em = 32px)

rem — Root Element Ke Relative

html { font-size: 12px; }
.paragraph { font-size: 2rem; } /* 2 * 12px = 24px */
  • rem hamesha <html> (root) ke font-size ke relative hota hai, parent ke nahi
  • Root 12px ho: 2rem = 24px, 3rem = 36px
  • Chahe kisi bhi parent element ka font-size change kar do, rem value change nahi hogi — sirf root font-size matter karta hai
⚠️ Yehi core difference hai: em = parent ke relative, rem = hamesha root (html) ke relative.

7. vmin / vmax

  • vmin — viewport ki chhoti side (width ya height, jo bhi smaller ho)
  • vmax — viewport ki badi side (jo bhi larger ho)

Phone par height zyada hoti hai width se → vmin = width, aur desktop par width zyada hoti hai → vmin = height.

.container {
  width: 100vmin; /* smaller dimension ka 100% */
}

Isse layout screen ke shape (portrait/landscape) ke hisaab se bhi adapt hota hai — sirf size hi nahi.

📝 Active Recall

Question par click karo — answer reveal hoga. Khud pehle try karo, phir check karo.

Absolute aur relative units mein basic farq kya hai?
Absolute units (px, pt, cm, mm) fixed values hoti hain, screen size se independent. Relative units (%, em, rem, vw, vh) kisi aur value (parent font-size, screen size) ke basis par calculate hoti hain.
Agar parent div 500px wide ho, to width: 50% wala box kitna wide hoga?
250px — kyunke percentage parent ki width ka relative calculate hoti hai.
Parent ki width change karne se width: 100px wala box aur width: 50% wala box, dono ka kya hota hai?
width: 100px wala box hamesha 100px hi rahega (fixed). width: 50% wala box automatically resize ho jaayega taake nayi parent width ka 50% bana rahe.
box-sizing: border-box kis problem ko solve karta hai?
Jab element ki fixed width ho aur usme border bhi ho, to border defined width se bahar add hoke unwanted horizontal scrollbar create kar sakta hai. border-box lagane se border aur padding width ke andar hi included ho jaate hain.
vw aur vh kis cheez ke relative hote hain?
Viewport (browser window) ki current visible width aur height ke relative — vw = viewport width, vh = viewport height.
Pixel-based fixed layout chhoti screen (jaise 1920px monitor) par kyun break ho sakta hai?
Kyunke pixel values fixed hoti hain aur screen size ke hisaab se adjust nahi hoti — agar defined pixel width screen se zyada ho, to layout overflow ho jaata hai ya break ho jaata hai.
margin: 20px auto; ka format kya represent karta hai?
Pehli value (20px) top aur bottom margin hai. auto left aur right margin hai jo element ko horizontally center kar deta hai.
margin: auto centering sirf kis type ke elements par kaam karti hai?
Sirf block-level elements par — inline elements is tarah center nahi hote.
em unit kis cheez ke relative calculate hota hai?
Apne immediate parent element ke font-size ke relative — 1em = parent ke font-size ka 100%.
Agar parent ka font-size 20px ho aur paragraph 2em use kare, to paragraph ka font-size kya hoga?
40px (2 × 20px).
Agar parent par koi font-size explicitly set na ho aur paragraph 2em use kare, to font-size kya hoga?
32px — kyunke browser ka default font-size 16px hota hai, aur 2 × 16px = 32px.
rem kis cheez ke relative hota hai?
Root element (html tag) ke font-size ke relative — parent ke font-size se koi lena dena nahi.
Agar html ka font-size 12px ho aur paragraph 3rem use kare, to font-size kya hoga? Agar parent ka font-size 20px kar diya jaaye to kya badlega?
3rem = 3 × 12px = 36px. Parent ka font-size 20px karne se koi farq nahi padega — paragraph 36px hi rahega, kyunke rem sirf root se calculate hota hai, parent se nahi.
em aur rem mein core difference ek line mein batao.
em parent element ke font-size ke relative hota hai, rem hamesha root (html) element ke font-size ke relative hota hai.
vmin aur vmax mein kya farq hai?
vmin = viewport ki chhoti dimension (width ya height, jo bhi smaller ho). vmax = viewport ki badi dimension (jo bhi larger ho).
Phone (portrait mode) par vmin generally kaunsi dimension ke barabar hota hai, aur desktop par?
Phone par height width se zyada hoti hai, isliye vmin = width. Desktop par width height se zyada hoti hai, isliye vmin = height.
