🖱️ CSS Overflow & Scrollbar Styling — Active Recall Notes

Video: overflow (auto/hidden/scroll), overflow-x/y, aur ::-webkit-scrollbar styling

1. Problem: Content Container Se Bada

Jab kisi element ka content uske box se bada ho, to wo overflow ho jaata hai. Isko control karne ke liye overflow property use hoti hai.

2. overflow: auto

.box {
  overflow: auto;
}

Agar content out-of-bounds ho, to scrollbars display ho jaate hain (sirf jitni zaroorat ho, horizontal ya vertical ya dono).

3. overflow: hidden

.box {
  overflow: hidden;
}

Overflowing content cut off ho jaata hai — bilkul dikhta hi nahi, koi scrollbar bhi nahi milti.

4. overflow: scroll

.box {
  overflow: scroll;
}

Scrollbars hamesha visible rehti hain — chahe content container mein fit hi kyun na ho jaaye.

5. overflow-x / overflow-y — Sirf Ek Direction Control Karna

Agar sirf ek specific axis (horizontal ya vertical) ka overflow control karna ho, to overflow-x aur overflow-y use hote hain:

.box {
  overflow-x: hidden; /* horizontal scrollbar gayab */
  overflow-y: hidden; /* vertical scrollbar gayab */
}
⚠️ Example: agar image ki width container se zyada ho (jaise 600px), to horizontal scrollbar aa jaata hai. overflow-x: hidden se sirf wo horizontal scrollbar hata sakte hain, vertical scroll par asar nahi padega.

6. Scrollbar Styling — ::-webkit-scrollbar

WebKit-based browsers (Chrome, Safari, Edge) mein scrollbar ko custom style karne ke liye special selectors use hote hain:

.box::-webkit-scrollbar {
  width: 10px;
}

.box::-webkit-scrollbar-track {
  background: gray;
}

.box::-webkit-scrollbar-thumb {
  background: purple;
}

.box::-webkit-scrollbar-thumb:hover {
  background: purple;
}
SelectorKya Style Karta Hai
::-webkit-scrollbarPoori scrollbar ki size (jaise width)
::-webkit-scrollbar-trackScrollbar ka background "track" (channel jismein thumb move karta hai)
::-webkit-scrollbar-thumbActual draggable "thumb" jo scroll amount show karta hai
::-webkit-scrollbar-thumb:hoverThumb ka style jab mouse usse hover kare

📝 Active Recall

Question par click karo — answer reveal hoga. Khud pehle try karo, phir check karo.

overflow: auto ka behavior kya hai?
Content agar container se bada/out-of-bounds ho, to sirf tab scrollbars display hote hain — na hone par scrollbar nahi dikhta.
overflow: hidden set karne se overflowing content ka kya hota hai?
Content cut off ho jaata hai — bilkul nazar nahi aata, aur koi scrollbar bhi provide nahi hota use dekhne ke liye.
overflow: scroll aur overflow: auto mein kya farq hai?
overflow: scroll mein scrollbars hamesha visible rehte hain, chahe content fit ho ya na ho. overflow: auto mein scrollbars sirf tab dikhte hain jab actually zaroorat ho.
Agar sirf horizontal overflow control karna ho (vertical ko touch kiye bagair), to kaunsi property use hogi?
overflow-x
Agar image container se zyada wide ho aur horizontal scrollbar hatana ho, to kya karenge?
overflow-x: hidden; set karenge — isse sirf horizontal scrollbar gayab hoga, vertical scroll par koi asar nahi padega.
::-webkit-scrollbar selector kya control karta hai?
Poori scrollbar ki general styling, jaise uski width (thickness).
::-webkit-scrollbar-track aur ::-webkit-scrollbar-thumb mein kya farq hai?
-track scrollbar ke background channel/path ko style karta hai jismein thumb move karta hai. -thumb wo actual draggable part hota hai jo scroll position show karta hai.
Thumb ka color mouse hover par change karne ke liye kaunsa selector use hoga?
::-webkit-scrollbar-thumb:hover
