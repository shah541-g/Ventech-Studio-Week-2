Video: CSS Box Model basics (margin, border, padding, content, box-sizing)

1. Box Model Kya Hai?

HTML ka har element ek box hota hai. CSS se hum us box ki properties customize kar sakte hain. Har box 4 parts se milkar bana hota hai:

  • Content Content — actual content (jaise text)
  • Padding Padding — content aur border ke darmiyan ki space
  • Border Border — box ke around ki line
  • Margin Margin — is box aur baaki boxes ke darmiyan ki space
Content Padding Border Margin

2. Code Example

div {
  margin: 40px;
  border: 2px solid red;
  padding: 20px;
}

Isse teeno containers ke darmiyan spacing (margin), red border, aur content ke around padding aa jaata hai.

3. Padding / Margin — Shorthand Values

Values Matlab
padding: 10px 20px; 1st = top & bottom, 2nd = left & right
padding: 10px 20px 30px; 1st = top, 2nd = left & right, 3rd = bottom
padding: 10px 20px 30px 40px; 1st = top, 2nd = right, 3rd = bottom, 4th = left (clockwise)

Margin bhi bilkul isi tarah kaam karta hai.

Specific side ke liye individual properties bhi use ho sakti hain:

margin-bottom: 20px;
margin-top: 20px;
margin-left: 40px;
padding-bottom: 15px;
padding-top: 15px;

4. Box Sizing

box-sizing ye decide karta hai ke box ki width kaise calculate hogi.

  • content-box (default) — width sirf content ki hoti hai; padding aur border alag se add hote hain (total width badh jaati hai)
  • border-box — width mein padding aur border included hote hain (box apni defined width se bahar nahi jaata)

Example: agar width 100px hai aur box-sizing border-box hai, to padding+border isi 100px ke andar fit hoga — box chhota dikhega.

CSS box model ke 4 parts kaunse hain?
Content, Padding, Border, Margin.
Padding aur margin mein kya farq hai?
Padding = content aur border ke darmiyan space. Margin = is box aur dusre boxes ke darmiyan space (outside border).
margin: 10px 20px 30px 40px; ka matlab kya hai?
Top = 10px, Right = 20px, Bottom = 30px, Left = 40px (clockwise order, top se start).
Agar sirf 2 values dein padding: 10px 20px;, to kya hoga?
1st value (10px) = top & bottom padding. 2nd value (20px) = left & right padding.
Kisi ek specific side (jaise sirf left) par margin dene ke liye kaunsi property use hogi?
margin-left. Isi tarah margin-top, margin-right, margin-bottom bhi hoti hain — aur padding ke liye bhi same pattern.
box-sizing ki default value kya hoti hai?
content-box — is mein width sirf content ki calculate hoti hai, padding aur border box ki total width mein add hoke usse bada kar dete hain.
border-box aur content-box mein kya farq hai?
border-box: defined width mein padding + border already included hote hain (box bahar nahi badhta). content-box: padding + border defined width ke alawa add hote hain (box overall bada ho jaata hai).
Agar width 100px ho aur box-sizing = border-box ho, aur usme padding + border bhi ho, to box screen par kaisa dikhega?
Box chhota dikhega — kyunke padding aur border ke liye alag se space nahi milta, wo 100px ke andar hi accommodate hote hain (content area apne aap shrink ho jaata hai).
