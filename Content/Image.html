🖼️ CSS Image Alignment — Active Recall Notes

Video: 4 tareeke image ko center/align karne ke — margin auto, text-align, float, background-image

Setup

<div class="container">
  <img src="pic.jpg" class="image">
</div>
.image {
  border: 1px solid black;
  width: 150px;
  height: 80px;
}

Note: agar width aur height ka ratio original image se match na kare, to image thori skew (stretch/squeeze) ho sakti hai.

1Method 1 — margin: auto (Image Ko Block Banakar)

.image {
  display: block;
  margin-left: auto;
  margin-right: auto;
}
  • margin-left: auto; akele → image ko right push kar deta hai
  • margin-left: auto; + margin-right: auto; dono ek sath → image horizontally center ho jaati hai
⚠️ Zaroori: image par pehle display: block; lagana padta hai, kyunke img by default inline element hai, aur inline elements par margin auto centering kaam nahi karti.

2Method 2 — text-align (Container Par)

.container {
  text-align: right; /* ya center, left */
}

Ye property image par nahi, uske parent/container par lagti hai. Kyunke image inline element hai, to wo text ki tarah container ke text-align se affect hoti hai.

  • text-align: right; → image right side mein
  • text-align: left; → image left side mein (default)
  • text-align: center; → image center mein

3Method 3 — float (Image Par)

.image {
  float: right; /* ya left */
}

float image ko us direction (left/right) ki taraf le jaata hai, aur baaki content us ke around wrap ho sakta hai — text/inline flow ke liye classic tareeka.

4Method 4 — background-image (CSS Se Image Add Karna)

HTML se <img> tag hata kar, image ko CSS ke through container ka background banaya ja sakta hai:

.container {
  background-image: url('path/to/image.jpg');
  width: 100%;
  height: 80px;
}
⚠️ Default behavior: background-image apne aap repeat (tile) ho jaati hai poore container mein — jab tak width/height set na ho aur repeat control na ki jaaye.

Repeat rokne ke liye:

.container {
  background-repeat: no-repeat;
}

Image ko poori tarah fit karne ke liye (cut off na ho):

.container {
  background-size: contain;
}

background-size: contain; — image ko defined width/height ke andar poori tarah fit kar deta hai, bina crop kiye.

Image ko position karne ke liye:

.container {
  background-position: center;
}
