Video: text-shadow aur box-shadow

1. text-shadow — Syntax

h1 {
  text-shadow: 1px 1px;
}

Kam se kam 2 numbers chahiye:

  • 1st value — horizontal offset (shadow left/right kitna move hoga)
  • 2nd value — vertical offset (shadow upar/neeche kitna move hoga)

Sample Text

  • Horizontal value badhane se (jaise 2, 5, 10, 50) shadow right ki taraf zyada move hota hai
  • Vertical value badhane se shadow neeche (down) ki taraf zyada move hota hai

2. Blur Radius (3rd Value — Optional)

h1 {
  text-shadow: 3px 5px 5px;
}
  • 3rd value = blur amount
  • Zyada blur (jaise 100px) shadow ko itna faila deta hai ke wo dikhna hi band ho jaata hai
  • Chhoti values (1–5px) usually ek acha, subtle blur effect dete hain

3. Color (4th Value — Optional)

h1 {
  text-shadow: 3px 5px 5px red;
}

Agar color na diya jaaye, to default text color ka hi shadow ban jaata hai.

4. Multiple Shadows (Comma-Separated)

text-shadow mein ek se zyada shadows comma se separate karke di ja sakti hain:

h1 {
  text-shadow:
    3px 5px 5px red,
    -3px -5px 5px blue;
}

Negative offset values (jaise -3px -5px) shadow ko upar aur left ki taraf push karti hain — isse ek doosri opposite-direction shadow ban jaati hai, jo different color se distinct dikhti hai.

Double Shadow

5. box-shadow — Non-Text Elements Ke Liye

box-shadow ka syntax/process bilkul text-shadow jaisa hi hai, lekin ye divs jaise non-text elements par use hota hai.

#box-1 {
  width: 100px;
  height: 100px;
  background-color: #eee;
  box-shadow: 3px 3px 5px;
}
  • 1st value — horizontal offset (3px)
  • 2nd value — vertical offset (3px)
  • 3rd value — blur radius (5px)
  • 4th value (optional) — color

Isse box par ek "3D pop" effect aa jaata hai — jaise box screen se thora uthta hua lag raha ho.

6. text-shadow vs box-shadow — Kab Kya Use Karein

PropertyKis Par Use Hoti Hai
text-shadowText elements (h1, p, span waghera)
box-shadowNon-text elements (div, button, card waghera)

text-shadow ke liye minimum kitni values chahiye, aur wo kya represent karti hain?
Kam se kam 2 values — pehli horizontal offset (left/right movement), dusri vertical offset (up/down movement).
text-shadow: 1px 1px; se shadow kis direction mein move hota hai?
1px right (horizontal offset) aur 1px down (vertical offset).
text-shadow ki 3rd value kya control karti hai?
Blur radius — shadow kitni blur (faili hui) dikhegi.
Agar blur value bohot zyada (jaise 100px) rakh di jaaye to kya hota hai?
Shadow itni zyada spread/faded ho jaati hai ke practically wo dikhna hi band ho jaati hai.
text-shadow ki 4th value kya hoti hai, aur na diye jaane par kya hota hai?
Color — agar color specify na kiya jaaye to default text color ka hi shadow use hota hai.
Ek text-shadow property mein multiple shadows kaise add karte hain?
Har shadow ko comma se separate karke — jaise text-shadow: 3px 5px 5px red, -3px -5px 5px blue;
Negative offset values (jaise -3px -5px) shadow ko kis taraf push karti hain?
Upar (up) aur left ki taraf — positive values ke opposite direction mein.
box-shadow, text-shadow se syntax ke lihaz se kitna different hai?
Bilkul same syntax/process hai (horizontal offset, vertical offset, blur, optional color) — bas box-shadow non-text elements (jaise div) par use hota hai, text-shadow text elements par.
box-shadow: 3px 3px 5px; lagane se element par kaisa visual effect aata hai?
Ek "3D pop" effect — element aisa lagta hai jaise wo screen se thora sa uthta hua (raised) ho.
