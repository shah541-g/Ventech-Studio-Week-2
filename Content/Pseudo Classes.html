1. Pseudo-class Kya Hai?

Pseudo-class ek keyword hai jo selector ke sath add hoti hai — ye element ke behavior ko us waqt modify karti hai jab wo ek specific state mein ho (jaise hover hona, click hona, waghera).

selector:pseudo-class {
  /* styles */
}

2. :hover

button:hover {
  font-size: 1.1em;
  background-color: green;
}

Jab mouse cursor element ke upar ho, tab styles apply hoti hain.

3. :active

button:active {
  background-color: red;
}

Jab element par mouse button currently click/hold ho raha ho (yani press-down state).

Upar wale button ko click-hold karke dekho — background red ho jaayega.

4. Link States — :link, :hover, :active, :visited

Anchor tags (<a>) ke 4 special states hote hain:

a:link    { color: lightblue; }  /* default, unvisited link */
a:hover   { color: skyblue; font-size: 1.1em; }
a:active  { color: darkblue; font-size: 1.1em; }
a:visited { color: gray; }       /* already click ho chuka link */
Pseudo-classKab Apply Hoti Hai
:linkNormal, unvisited link state
:hoverMouse link ke upar ho
:activeLink click-hold ho rahi ho
:visitedUser pehle is link par click kar chuka ho (default purple ki jagah)

5. :not()

li:hover {
  background-color: yellow;
}

li:not(:hover) {
  color: gray;
}

:not() ka matlab: "jo is state mein NAHI hai" — yani opposite condition select karta hai. Yahan: jo list item currently hover nahi ho raha, uska color gray rahega.

6. :nth-child()

li:nth-child(1) { color: yellow; } /* 1st element */
li:nth-child(3) { color: yellow; } /* 3rd element */
li:nth-child(9) { color: yellow; } /* 9th element */

Parentheses ke andar value kaunsa child element select hoga wo decide karti hai (counting 1 se start hoti hai).

ValueKya Select Karta Hai
oddSab odd-numbered elements (1st, 3rd, 5th...)
evenSab even-numbered elements (2nd, 4th, 6th...)
3nHar teesra element (3rd, 6th, 9th...)
4nHar chautha element (4th, 8th, 12th...)
3n + 1Har teesra element, lekin 1st se start (1st, 4th, 7th...)
3n + 2Har teesra element, 2nd se start (2nd, 5th, 8th...)
⚠️ nth-child JavaScript ke sath bhi bohot handy hai — parentheses ke andar ki value ko ek variable se dynamically control kiya ja sakta hai.

7. hover Se Hidden Content Reveal Karna (Dropdown Trick)

.greeting p {
  display: none;
}

.greeting:hover p {
  display: block;
}

Ye trick dropdown menus banane ke liye bohot common hai — jab tak hover na ho, content display: none (poori tarah hidden aur layout se bahar) rehta hai. Hover hote hi display: block ho jaata hai aur content dikh jaata hai.

Pseudo-class kya hoti hai?
Ek keyword jo selector ke sath add hoti hai (colon ke sath) aur element ka behavior modify karti hai jab wo kisi specific state mein ho, jaise hover ya click.
:hover kis condition mein trigger hoti hai?
Jab mouse cursor us element ke upar ho.
:active kis condition mein trigger hoti hai?
Jab element par mouse button currently press/hold kiya ja raha ho (click-and-hold state).
Anchor tags ke 4 special pseudo-class states kaunse hain?
:link (default/unvisited), :hover (mouse upar), :active (click-hold), :visited (pehle click ho chuka link).
:visited pseudo-class kis liye use hoti hai?
Un links ka styling change karne ke liye jinhe user pehle hi click kar chuka ho (by default browser inhe purple dikhata hai).
:not() pseudo-class kya karta hai?
Opposite condition select karta hai — jo elements specified state mein NAHI hain, unhe target karta hai. Jaise li:not(:hover) un list items ko select karega jo currently hover nahi ho rahe.
nth-child() mein counting kis number se start hoti hai?
1 se — pehla element nth-child(1) hota hai.
nth-child(odd) aur nth-child(even) kya select karte hain?
odd: sab odd-numbered elements (1st, 3rd, 5th...). even: sab even-numbered elements (2nd, 4th, 6th...).
nth-child(3n) kya select karta hai?
Har teesra element — 3rd, 6th, 9th, waghera.
nth-child(3n + 1) ka behavior nth-child(3n) se kaise different hai?
3n har teesra element select karta hai starting from 3rd (3, 6, 9...). 3n + 1 offset add karta hai, isliye ye 1st se start hokar har teesra element select karta hai (1, 4, 7...).
Dropdown menu jaisa hidden-content-on-hover effect banane ke liye kaunsa combination use hota hai?
Content par display: none; (default hidden) aur parent element par :hover ke through us content ko display: block; kar dena.
nth-child() ki value JavaScript ke sath kyun handy hoti hai?
Kyunke parentheses ke andar ki value ko ek variable se dynamically set/control kiya ja sakta hai, jisse selection JS logic ke hisaab se change ho sakti hai.
